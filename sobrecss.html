CSS - Cascading Style Sheets

Seletores

As tags são os seletores a serem chamados no CSS, assim como Classe e ID

Seletores Tipo

  • h1
  • p
  • a
  • img

Seletores Multiplos

h2, h3, ol

Inline

O CSS inline é a estilização feita na própria linha do HTML para definir um estilo apenas naquela tag, linha ou elemento por meio do atributo style aberto dentro da tag que deseja estilizar

Interno

O CSS interno é feito dentro da tag HEAD abrindo a tag style

Externo

O CSS externo é feito em um documento separado do html e é chamado dentro da tag Head

Atribuir Classe

Classes podem ser chamada com ponto + seu nome, nome definido pela sintaxe dentro da tag: classe="nome"

Atribuir Id

Id é um identificador que pode ser chamado com # + nome
